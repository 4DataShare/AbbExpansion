
Each file correspondes to a subject project.

Columns contained in the dataset:

(1)id
(2)files			// paths where the identifier appears
(3)typeOfIdentifier		// type of identiifer 
(4)subclass			// names of its direct subclasses      
(5)subsubclass			// names of all subclasses of the direct subclasses
(6)parents			// names of its direct parents
(7)ancestor			// names of all parents of the direct parents
(8)methods			// method names of a class
(9)fields			// field names of a class  
(10)comment			// comments for the identifier
(11)type			// data type of the identifier
(12)enclosingClass		// names of its enclosing class
(13)assignment			// identifiers at the other side of assignment
(14)methodInvocated		// names of invoated method
(15)parameterArgumnt	        // names of corresponding parameters
(16)parameter			// parameter names of a method
(17)enclosingMethod		// names of the enclosing method
(18)argument			// names of corresponding arguments
(19)name			// name of the identifier
(20)abbr			// abbreviation in the identifier
(21)expansion			// full expansion
